# MojoDas Spa

Lithuanian luxury hot tub e-commerce platform.

## 🚀 Quick Start

```bash
./start.sh
```

Or manually:

```bash
cd mojodas-spa
pnpm install
pnpm dev
```

Visit: **http://localhost:3000**

## 📁 Structure

```
mojodasspa/
├── start.sh              # Quick launch script
├── mojodas-spa/          # Next.js 16 application
│   ├── public/images/    # 112 product images (45MB)
│   ├── data/             # Product catalog JSON
│   └── app/              # Next.js App Router pages
├── PRD.md                # Product requirements
└── IMPLEMENTATION_PLAN.md
```

## 🛠️ Tech Stack

- **Next.js 16.2.1** (App Router, React 19)
- **Tailwind CSS v4**
- **Zustand** (State management)
- **TypeScript** (Strict mode)
- **Lithuanian** language throughout

## 📦 Features

- 16 hot tub products with full specs
- Visual product configurator (60+ options)
- Shopping cart & checkout
- B2B wholesale portal
- Contact forms with email integration
- Responsive design

## 🌐 Deploy

```bash
cd mojodas-spa
vercel --prod
```
